## 📋 Overview

**Hiring Process Analytics** is a data-driven analysis project that examines historical hiring records to uncover insights about recruitment patterns, compensation structures, and workforce composition. This project was developed as a lead Data Analyst role at a major multinational corporation, utilizing advanced statistical analysis and Excel-based data manipulation to extract meaningful conclusions about the company's hiring processes.

The project demonstrates practical application of business analytics, statistics, and data visualization techniques to support human resources decision-making and strategic planning.

## ✨ Key Features

- **👥 Demographic Analysis**: Comprehensive breakdown of male/female hiring ratios
- **💰 Salary Statistics**: Average salary calculation and distribution analysis
- **📈 Class Intervals**: Detailed salary range classification and frequency distribution
- **📊 Department Distribution**: Visualization of workforce across different departments
- **🏢 Job Tier Analysis**: Analysis of hierarchical position levels in the organization
- **📉 Statistical Insights**: Use of advanced statistical formulas and methodologies
- **📋 Excel-Based Analytics**: All analysis performed using Excel/Google Sheets formulas
- **🎨 Data Visualization**: Professional charts and graphs for easy interpretation

## 🎯 Analysis Questions & Objectives

### 1. **Gender Distribution in Hiring**
**Question**: How many males and females are hired?

**Analysis Includes**:
- Total count of male employees
- Total count of female employees
- Gender ratio calculation
- Percentage distribution
- Hiring bias analysis (if any)

**Methods Used**:
- COUNTIF() function for gender categorization
- Simple percentage formulas
- Gender distribution visualization

### 2. **Average Salary Analysis**
**Question**: What is the average salary offered in this company?

**Analysis Includes**:
- Overall average salary calculation
- Average salary by gender
- Average salary by department
- Average salary by job tier
- Salary trends and patterns

**Methods Used**:
- AVERAGE() function
- AVERAGEIF() for conditional averages
- Statistical measures (mean, median, mode)
- Salary benchmarking

### 3. **Salary Distribution - Class Intervals**
**Question**: Draw class intervals for salary distribution in the company?

**Analysis Includes**:
- Salary range classification (bins/intervals)
- Frequency distribution of salary ranges
- Cumulative frequency analysis
- Distribution histogram
- Percentile analysis

**Methods Used**:
- FREQUENCY() function
- Bin creation and grouping
- Cumulative sum formulas
- Distribution analysis
- Histogram visualization

### 4. **Department Distribution - Charts & Plots**
**Question**: Show proportion of people working in different departments?

**Analysis Includes**:
- Count of employees per department
- Percentage distribution by department
- Department-wise salary analysis
- Department-wise gender ratio
- Department composition visualization

**Methods Used**:
- COUNTIF() for department counts
- PIE CHARTS for visual representation
- BAR GRAPHS for comparison
- Data labels and percentages

### 5. **Job Tier Representation - Charts**
**Question**: Represent different post tiers using chart/graph?

**Analysis Includes**:
- Hierarchy levels in organization
- Count of positions at each tier
- Salary progression by tier
- Career path visualization
- Tier-wise distribution analysis

**Methods Used**:
- Bar charts for tier comparison
- Stacked bar charts for composition
- Salary range by tier visualization
- Organizational hierarchy representation
